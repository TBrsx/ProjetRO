Tristan Bersoux, Nicolas Compère, 2021
Projet de fin d'UE Recherche Opérationelle

Tournées de véhicules dans plusieurs villes afin de fournir des ressources, on veut minimiser la distance totale parcourue.

Instructions d'utilisation :
Dans le REPL Julia, importer le fichier main.jl, lancer la fonction solveCovid19(nomFichier) avec nomFichier étant le fichier contenant les données du problème.
Pour des tests plus approfondis, on peut importer le fichiers test.jl et utiliser la fonction de test (Em modifiant la variable n dans le code à sa guise).
